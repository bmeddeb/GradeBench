How to Generate a Canvas API Key

  1. Log in to your Canvas account
  2. Go to Account > Settings
  3. Scroll down to "Approved Integrations"
  4. Click "New Access Token"
  5. Enter a purpose (e.g., "GradeBench Integration")
  6. Set an expiration date (or leave blank for no expiration)
  7. Click "Generate Token"
  8. Copy the token and paste it here

Note: The token will only be displayed once. If you lose it, you'll need to generate a new one.
